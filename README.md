  # Velocity-Labs
  AI-powered revenue simulation and growth analytics platform.
  ## Features
- 4 simulation modes: Startup, Creator, E-commerce, SaaS
- 12-month revenue projections with 3 scenarios
- Analytics dashboard with funnel, LTV/CAC, and goal tracking
- USD / INR currency toggle
- ## Tech Stack
- HTML / CSS / JavaScript (no frameworks)
- Custom canvas chart engine (no Chart.js)
- localStorage for client-side persistence
